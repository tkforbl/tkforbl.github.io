1. 개인정보 수집 및 이용 목적

AVC 앱은 다음과 같은 목적으로 개인정보를 수집하고 이용합니다:

  • 앱 기능 제공: 앱별 볼륨 제어 기능을 위한 설정 정보
  • 오류 개선: 앱 안정성 향상을 위한 오류 보고 정보
  • 서비스 개선: 사용자 의견을 통한 서비스 개선

2. 수집하는 개인정보 항목

  • 개인식별정보, 개인민감정보: 미수집
  • 앱 설정 정보 (총 13개 항목):
    • 볼륨 UI 표시 타입, 좌표(X,Y), 크기 설정
    • 수동 볼륨 변경 시 표시 여부
    • 앱 감지 시 이름 표시 여부
    • 가로 모드 좌표 및 별도 설정 사용 여부
    • 폰트 크기 및 오버레이 표시 시간
    • 볼륨 표시 모드 설정
    • 디바운싱 활성화 및 지연 시간 설정
  • 오류 로그: 앱 실행 중 발생하는 오류 및 예외 정보
  • 기기 정보: 앱 버전, 기기 모델, Android 버전 등
  • 사용자 의견: 오류 보고 시 입력하는 추가 설명 내용

3. 개인정보 수집 방법

  • 자동 수집: 앱 사용 중 자동으로 수집되는 정보
  • 수동 입력: 사용자가 직접 입력하는 설정 및 의견
  • 권한 기반: 사용자 동의 하에 수집되는 정보

4. 개인정보 보유 및 이용 기간

  • 앱 설정 정보: 앱 삭제 시까지
  • 오류 로그: 오류 해결 완료 후 30일 이내 삭제
  • 사용자 의견: 의견 처리 완료 후 1년 이내 삭제
  • 기기 정보: 앱 삭제 시까지

5. 개인정보 제3자 제공

AVC 앱은 다음과 같은 경우에만 개인정보를 제3자에게 제공합니다:

  • 사용자 동의가 있는 경우
  • 법령에 의해 요구되는 경우
  • 오류 분석을 위해 Firebase Crashlytics에 익명화된 정보 제공

6. 개인정보 보호 조치

  • 암호화: 중요 정보는 암호화하여 저장
  • 접근 제한: 개인정보에 대한 접근 권한 제한
  • 안전한 전송: HTTPS를 통한 안전한 데이터 전송
  • 정기 점검: 개인정보 보호 상태 정기 점검

7. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 개인정보 처리정지 요구
  • 개인정보 이전 요구
